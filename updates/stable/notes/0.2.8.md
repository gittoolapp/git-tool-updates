# Git Tool 0.2.8

## Packaging

- Added the Git Tool app icon to packaged builds and named the macOS bundle
  `Git Tool.app`.

## Fixes And Reliability

- Moved the repository switcher, environment switcher, and error center popovers
  onto the shared fixed floating layer, so dropdown content no longer renders
  underneath sidebar filters, branch rows, or other panel content on affected
  systems.
- Kept the repository switcher dropdown aligned with the sidebar edge and made
  repository/environment search fields focus reliably when their dropdowns open.
- Marked shared floating popovers as non-draggable Wails UI so portaled
  dropdown search fields can receive focus in the desktop window.
- Fixed macOS in-app updates across the bundle rename from `git-tool.app` to
  `Git Tool.app`, so the helper backs up the old bundle, installs the renamed
  bundle, and relaunches the updated app instead of leaving two apps side by
  side.
- Stash pop/apply conflicts now open the conflict workspace instead of leaving
  the app in normal history mode with conflicted files in the working tree.
