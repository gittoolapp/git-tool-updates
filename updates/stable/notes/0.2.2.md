# Git Tool 0.2.2

## UI Polish

- Removed the visible borders from commit graph branch and tag reference chips,
  made the chips taller, and set the commit graph row height to 30px so row text
  and lane markers read larger.

## Settings And Updates

- Changed available updates to a single **Install and Restart** action that
  downloads, verifies, prepares, installs, and restarts internally without
  exposing a separate patch-download step.
- Removed patch-file management from the normal Settings flow and clears the
  cached patch/full artifact from the update cache after the staged install
  payload is prepared.
